# Personal Portfolio Website

Built with **Next.js**, **Tailwind CSS**, and **Framer Motion**.

## Features
- **Project Showcase**: Auto-generated from JSON data.
- **Blog**: Placeholder for future posts.
- **Contact Form**: Serverless API route.
- **Responsive Design**: Mobile-first approach.
- **E2E Testing**: Cypress integration.

## Getting Started

```bash
npm install
npm run dev
```
